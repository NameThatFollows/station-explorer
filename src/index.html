Transit Map Challenge

How many rail stations have you been to? Select a map on the left and click stations you've been to on the map or use the station list on the right.

Rules

  • You must enter and/or exit the station in order for it to count.
    • Passing through does not count.
    • Transferring to another line does not count.
  • In the spirit of the challenge, you should explore the neighborhood around the station.

Notes

  • If a station appears on multiple maps, checking that station will check it on all maps.
  • If a map shows connecting stations from other systems, those stations will also be clickable and will show up in a separate section in the stations list.

About

If you would like to request a new map, report a bug, or submit general feedback, please create an issue.
